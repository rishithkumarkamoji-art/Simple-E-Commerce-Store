# Market Lane E-commerce Site

A basic full-stack e-commerce app built with HTML, CSS, JavaScript, Express.js, and SQLite.

## Features

- Product listings seeded into SQLite
- Product details page
- User registration and login
- Persistent shopping cart per user
- Checkout and order processing
- Order history

## Run

```bash
npm install
npm start
```

Then open `http://localhost:3000`.

The SQLite database is created automatically at `data/shop.sqlite`.
